# Development

## Prerequisites

- Init submodules:

```
git submodule update --init --recursive
```

### Running locally

- Install [Hugo](https://gohugo.io/installation/)
- Run the hugo server:

```
hugo server -D -s src/
```

### Running in docker

- Install [docker-compose](https://docs.docker.com/compose/install/)
- Run the hugo server:

```
docker-compose up
```
